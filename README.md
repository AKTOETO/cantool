
# CanGUI

Графический интерфейс для работы с CAN-шиной.

## Описание

CanGUI — это Python-приложение для мониторинга и управления CAN-шиной с графическим интерфейсом.

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
python main.py
```
